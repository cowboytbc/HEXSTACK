# Choose your OS first

This bundle contains two installer packages:

- `Windows/` for Windows users
- `macOS/` for Mac users

## Windows

Open the `Windows/` folder and run:

- `Install-HEXSTACK-Windows.ps1`

## macOS

Open the `macOS/` folder and run:

- `Install-HEXSTACK-Mac.command`

Each installer helper lets the user choose what to install and where to install it.
